# Antigravity 에이전트 규칙 (Project-Specific)

에이전트는 이 프로젝트 내에서 작업을 수행할 때 반드시 전역 규칙([GEMINI.md](file:///C:/Users/admin/.gemini/GEMINI.md))을 기본으로 따르며, 추가로 아래의 프로젝트 특화 규칙을 준수해야 합니다.

## 1. 전역 규칙 계승

- 모든 답변은 한국어로 작성합니다.
- 명시적 지시 없는 독단적인 코드 수정 및 질문을 즉시 수정으로 연결하는 행위를 금지합니다.
- 프로젝트 관리 문서(`main_plan.md`, `main_task.md`, `main_walkthrough.md`)를 필수적으로 유지 및 업데이트합니다.

## 2. 프로젝트 특화 문서 관리 (Core)

- **문서 저장 경로**: 모든 프로젝트 관리용 메인 문서는 반드시 **`/.gemini/docs/`** 경로에서만 관리합니다.
- **Single Source of Truth**: `/.gemini/docs/` 폴더 내의 파일들이 프로젝트의 현재 상태를 나타내는 유일한 공식 문서입니다.
- **문서화 예외 사항 (중요)**:
  - **`.agentignore`** 또는 **`.claudeignore`**에 명시된 폴더와 관련된 작업 시에는 물리적인 `.md` 파일 생성을 생략합니다.
  - 이 경우, 에이전트 내부 UI(Task view)를 통해서만 작업 절차를 진행하고 보고합니다.

## 3. 에이전트 설정

- 이 프로젝트의 모든 워크플로우 및 추가 지침은 `.gemini/` 디렉토리 내에서 관리합니다.

## 4. 규칙 우선순위

- 이 파일(`.gemini/rules.md`)에 정의된 규칙은 전역 설정보다 우선하며, 에이전트의 모든 도구 호출 및 의사결정에 반영되어야 합니다.
