# [스킬 명칭] (예시: README 관리 스킬)

## 개요

이 스킬은 프로젝트의 README.md 파일을 최신 상태로 유지하고 한국어로 번역하는 작업을 지시합니다.

## 실행 지침

1. `package.json` 버전이 업데이트되면 README의 버전 정보를 동기화합니다.
2. 새로운 기능이 추가되면 '기능 목록' 섹션에 내용을 추가합니다.
3. 영문 README가 있다면 반드시 동일한 내용의 `README_KR.md`를 생성하거나 업데이트합니다.

## 주의 사항

- 기술 용어는 괄호를 사용하여 영문을 병기합니다. (예: 가상화(Virtualization))
