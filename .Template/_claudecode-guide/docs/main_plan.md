# Wikidocs '클로드 코드 가이드' 복사 구현 계획

Wikidocs 북 [클로드 코드 가이드](https://wikidocs.net/book/19104)의 전체 내용을 동일한 계층 구조의 Markdown(.md) 파일로 로컬에 복사하는 것을 목표로 합니다.

## 제약 사항 및 전략

- **브라우저 사용**: 직접적인 HTML 파싱(403 에러)이나 자동화된 서브에이전트 용량 제한으로 인해, 브라우저 도구를 사용하여 페이지별로 접근 후 내용을 텍스트 형식으로 추출합니다.
- **구조 유지**: 목차의 계층(Chapter > Lesson)을 폴더 및 파일명에 반영합니다.

## 제안된 변경 사항

### [New Project Structure]

Wikidocs의 목차 구조에 따라 다음과 같은 디렉토리와 파일을 생성합니다.

#### [NEW] docs/

- `main_plan.md`: 전체 작업 계획
- `main_task.md`: 상세 진행 단계
- `main_walkthrough.md`: 최종 결과 요약

#### [NEW] content/ (또는 루트)

- `01_README.md`
- `02_n8n_설치가이드/`
  - `02-01_n8n_클라우드.md`
  - ...
- `05_AI_Agent/`
  - `05-01_model/`
    - `05-01-01_OpenAI.md`
    - `05-01-02_Claude.md`
- ... (전체 목차 반영)

## 검증 계획

### 수동 검증

1. **파일 구조 확인**: 생성된 폴더와 파일 목록이 Wikidocs 목차와 일치하는지 `ls -R` 명령어로 확인합니다.
2. **내용 비교**: 무작위로 3~5개 파일을 선택하여 원본 웹페이지의 내용(텍스트, 코드 블록)이 누락 없이 Markdown으로 변환되었는지 확인합니다.
