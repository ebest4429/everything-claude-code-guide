# 10. 서브에이전트

복잡한 작업을 분담하기 위해 전용 컨텍스트를 가진 하위 에이전트를 생성합니다.

## 특징

- **독립적 컨텍스트**: 메인 세션과 분리된 히스토리와 메모리를 가집니다.
- **병렬 처리**: 여러 서브에이전트가 동시에 다른 작업을 수행할 수 있습니다.
- **특화된 역할**: 특정 도구나 권한만 부여하여 보안성과 효율성을 높입니다.

## 실행 모드

- **Foreground**: 에이전트의 작업 과정을 실시간으로 관찰
- **Background**: 백그라운드에서 작업을 수행하고 완료 시 보고

## 워크트리 격리

복잡한 수정 사항의 경우 별도의 Git 워크트리를 생성하여 메인 코드베이스에 영향을 주지 않고 안전하게 작업할 수 있습니다.
