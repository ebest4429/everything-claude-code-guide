# 11. 스킬 (Skills)

자주 반복되는 복잡한 프롬프트나 작업 시퀀스를 이름이 지정된 '스킬'로 저장합니다.

## 스킬 파일 구조

`.claude/skills/` 디렉토리에 `.md` 파일 형식으로 저장됩니다.

## 구성 요소

- **Frontmatter**: 스킬 이름, 설명, 매개변수 정의
- **Content**: Claude에게 전달할 구체적인 지침(System Prompt 스타일)

## 호출 제어

슬래시 명령어 등을 통해 스킬을 명시적으로 실행하거나, Claude가 상황에 맞춰 적절한 스킬을 스스로 선택하여 사용할 수 있습니다.
