# 05. 권한 시스템

**권한 관리 개요**

- 클로드 코드는 도구 사용 전 반드시 사용자의 승인을 거칩니다.
- `settings.json`의 `permissions` 항목을 통해 자동 승인(`allow`) 및 명시적 거부(`deny`)를 설정할 수 있습니다.

**보안 매커니즘**

- **Deny 우선**: 허용 규칙보다 거부 규칙이 항상 우선 적용됩니다.
- **샌드박싱**: `sandboxPathPrefix`를 설정하여 접근 범위를 제한할 수 있습니다.
- **PreToolUse 훅**: 도구 실행 전 사용자 정의 스크립트 실행.
