# 12. IDE 연동

Claude Code를 선호하는 개발 환경과 통합하여 생산성을 극대화합니다.

## 지원 환경

1. **VS Code**: 전용 확장을 통해 터미널과 에디터를 긴밀하게 연결
2. **JetBrains (IntelliJ, WebStorm 등)**: 플러그인을 통한 통합 지원
3. **데스크톱 앱**: 독립적인 실행 환경 제공

## 주요 기능

- **에디터 자동 스크롤**: Claude가 수정하는 파일 위치로 자동 이동
- **권한 모드 선택**:
  - `Ask`: 매 작업마다 승인 요청
  - `Auto-accept`: 파일 수 자동으로 승인
  - `Plan`: 코드 분석 및 계획만 수립하고 수정은 보류
- **프리뷰 서버**: `.claude/launch.json` 설정을 통해 개발 서버 구동 및 자동 검증
