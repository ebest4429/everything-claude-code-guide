# 별첨 1. 클로드 코드 창시자의 워크플로우

Claude Code 개발팀(창시자들)이 직접 공개한 '프로처럼 사용하는 법'입니다.

## 핵심 전략

- **작업 분할**: 하나의 큰 요구사항을 여러 개의 작은 태스크로 쪼개어 서브에이전트에게 할당.
- **지속적 피드백**: 중간 결과물을 짧게 확인하고 방향을 수정하는 인터랙티브 방식 권장.

## 프롬프트 팁

- 구체적인 컨텍스트 제공: "이 파일의 10라인부터 50라인 사이의 로직만 수정해줘"와 같은 명시적 지시.
- 제약 조건 활용: 특정 라이브러리 사용 금지 또는 성능 요구사항 명시.

## 도구 활용

- 커스텀 훅을 이용한 본인만의 자동화 스크립트 결합.
- MCP 서버를 활용한 통합 검색 환경 구축.
