# 19. 실전 워크플로우

실무에서 Claude Code를 100% 활용하는 구체적인 시나리오와 전략입니다.

## 새 코드베이스 이해하기

- **Explore Codebase**: `grep`과 `find` 도구를 조합하여 전체 구조를 빠르게 파악.
- **관련 코드 탐색**: 질문을 통해 특정 기능이 구현된 파일과 함수를 자동 검색.

## 효율적인 작업 흐름

- **버그 수정**: `Plan 모드`로 수정을 계획하고, `Subagent`에게 세부 수정을 맡김.
- **리팩토링**: 대규모 변경 시 `Git Worktree`를 생성하여 메인 브랜치와 격리된 환경에서 작업.

## 고급 기능 활용

- **이미지 분석**: 스크린샷이나 UI 기획서를 첨부하여 레이아웃 구현 요청.
- **Extended Thinking**: 복잡한 논리 설계 시 더 깊은 추론 모드 활성화.
- **알림 설정**: 장시간 소요되는 작업 완료 시 시스템 알림(OS별 지원) 수신.

## 테스트 및 PR

- 변경 사항에 대한 유닛 테스트 자동 실행.
- 작업 내용을 바탕으로 GitHub Pull Request 설명문(Description) 자동 생성.
