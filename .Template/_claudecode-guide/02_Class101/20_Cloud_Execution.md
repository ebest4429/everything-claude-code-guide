# 20. 클라우드 실행 (웹)

터미널 설치 없이 웹 브라우저에서 직접 Claude Code를 사용하는 방법입니다.

## 웹 환경 구축

- **claude.ai 접속**: 웹 인터페이스를 통한 상호작용.
- **GitHub 연동**: 자신의 저장소를 클라우드 환경에 연결하여 즉시 작업 시작.

## 동작 및 UI

- **가상 머신(VM)**: 클라우드 기반의 격리된 환경에서 코드 실행.
- **Diff 뷰**: 코드 변경 사항을 웹 화면에서 시각적으로 검토 및 승인.

## 보안 및 정책

- **네트워크 보안**: 외부 통신이 차단된 안전한 샌드박스 환경.
- **사용량 제한**: 가격 정책에 따른 일일 호출 한도 및 구독 모델 안내.
