# 15. 고급 기능

Claude Code의 성능을 극대화하는 심화 활용 기술입니다.

## 복합 워크플로우

- **에이전트 협업**: 대규모 작업을 여러 에이전트에게 분배하는 전략.
- **컨텍스트 최적화**: 토큰 사용량을 줄이면서 정확도를 높이는 방법.

## 멀티모달 활용

- **이미지 처리**: CLI 환경에서 스크린샷이나 다이어그램을 분석하는 기능 활용.
- **데이터 분석**: 대량의 로그 파일을 시각화하거나 구조화하여 분석.

## 성능 튜닝

- 시스템 프롬프트 조정을 통한 일관된 출력 확보.
- 자주 사용하는 도구의 사전 로딩 설정.
