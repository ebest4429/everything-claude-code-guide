# 21. 보안과 프라이버시

사용자의 데이터와 코드를 보호하기 위한 안트로픽의 보안 정책입니다.

## 데이터 보호 (ZDR)

- **Zero Data Retention**: 사용자의 입력 데이터와 소스 코드는 모델 학습에 사용되지 않습니다.
- **데이터 격리**: 각 세션은 독립적으로 관리되며 세션 종료 후 휘발성 데이터는 삭제됩니다.

## 아키텍처 보안

- **종단간 암호화**: 모든 통신은 최신 암호화 프로토콜을 통해 보호됩니다.
- **프롬프트 보안**: 악의적인 프롬프트 인젝션 및 무단 도구 실행 방어.

## 로컬 보안

- **권한 제어**: 파일 읽기/쓰기 및 명령 실행 전 항상 사용자 승인 필요.
- **환경 변수 보호**: API 키 등 민감 정보의 안전한 저장 및 노출 방지.
