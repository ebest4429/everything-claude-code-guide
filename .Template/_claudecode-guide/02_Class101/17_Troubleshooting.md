# 17. 트러블슈팅

Claude Code 사용 중 발생할 수 있는 문제와 해결 방법입니다.

## 일반적인 오류 해결

- **네트워크 이슈**: 타임아웃 또는 연결 거부 시 프록시 및 방화벽 확인.
- **권한 오류**: `permission denied` 발생 시 설정 파일의 권한 섹션 검토.
- **제한(Quota)**: API 호출 한도 초과 시 티어 업그레이드 또는 사용량 조절.

## 디버깅 도구

- `--verbose`: 내부 통신 로그를 출력하여 상세 원인 파악.
- `/version`: 현재 설치된 버전 확인 및 업데이트 필요성 체크.

## 플랫폼별 특이사항

- **Windows (WSL)**: 파일 시스템 경로 호환성 문제 처리.
- **macOS/Linux**: 권한 비트 및 환경 변수 상속 이슈.
