# 도로명정보조회 엑셀 통합 (프롬프트 전용)

## 입력 파싱

`$ARGUMENTS`에서 시도명과 폴더경로를 파싱합니다.

- 첫 번째 토큰 = 시도명 (예: 충남, 전북, 경기)
- 두 번째 토큰 = 폴더경로 (예: /Users/me/충남_도로명)

둘 중 하나라도 없으면 사용자에게 질문 후 진행합니다.

## 처리 규칙

각 원본 xlsx 파일의 구조와 처리 방법:

```
Row 0  →  "총N건 (도로명수 : N)"   ← 모든 파일에서 제거
Row 1  →  헤더 행                  ← 첫 번째 파일에서만 포함
Row 2+ →  실제 데이터              ← 모든 파일에서 포함
```

## 실행 절차

1. 폴더 내 xlsx 파일 목록 확인 (출력 파일 자신은 제외)
2. pandas로 각 파일을 읽어 위 규칙 적용
3. 전체 데이터를 하나의 DataFrame으로 합치기
4. openpyxl로 서식 포함 저장:
   - 헤더: 파란 배경(#4472C4) + 흰색 굵은 글씨
   - 데이터: Arial 10pt, 테두리 적용
   - 1행 틀 고정
5. 저장 경로: `[폴더경로]/[시도명]_도로명정보조회.xlsx`

## 완료 후 보고

- 생성된 파일 경로 안내
- 총 데이터 행 수 및 파일 수 요약
- 오류 발생 시 원인과 해결 방법 설명
