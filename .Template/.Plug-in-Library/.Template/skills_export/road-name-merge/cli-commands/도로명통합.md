# 🗺️ 도로명정보조회 엑셀 통합

## 📖 사용법 (Usage)

```
/도로명통합 [시도명] [폴더경로]
```

**예시:**
```
/도로명통합 충남 /Users/me/Documents/충남_도로명
/도로명통합 전북 C:/Users/me/Desktop/전북_도로명
/도로명통합 경기 ~/Downloads/경기_도로명
```

**파라미터:**
- `[시도명]` — 출력 파일명 앞에 붙는 시도 이름 (예: 충북, 충남, 전북, 경기)
- `[폴더경로]` — 병합할 xlsx 파일들이 들어있는 폴더 경로

**출력 파일:** `[폴더경로]/[시도명]_도로명정보조회.xlsx`

---

## ⚙️ 처리 규칙

각 엑셀 파일의 구조:
```
Row 0  →  총N건 (도로명수 : N)   ← 전체 파일 제외
Row 1  →  헤더 (시군구, 도로명번호 …) ← 첫 번째 파일만 포함
Row 2+ →  실제 데이터              ← 전체 파일 포함
```

---

## 🚀 실행

$ARGUMENTS 에서 시도명과 폴더경로를 파싱합니다.

아래 단계를 순서대로 진행합니다.

### Step 1 — 입력값 파싱
```python
args = "$ARGUMENTS".strip().split()
# args[0] = 시도명, args[1] = 폴더경로
```
인자가 부족하면 사용자에게 시도명과 폴더경로를 질문합니다.

### Step 2 — 폴더 내 xlsx 파일 확인
```bash
ls "[폴더경로]"/*.xlsx
```
파일이 없으면 경로를 다시 확인하도록 안내합니다.

### Step 3 — 병합 실행

아래 Python 코드를 실행합니다 (인라인 실행):

```python
import os, sys, pandas as pd
from openpyxl import Workbook
from openpyxl.styles import Font, PatternFill, Alignment, Border, Side
from openpyxl.utils import get_column_letter

def merge(folder_path, province_name):
    folder_path = os.path.abspath(folder_path)
    output_filename = f"{province_name}_도로명정보조회.xlsx"
    output_path = os.path.join(folder_path, output_filename)

    files = sorted([f for f in os.listdir(folder_path)
                    if f.endswith(".xlsx") and f != output_filename])
    if not files:
        raise FileNotFoundError(f"'{folder_path}'에서 xlsx 파일을 찾을 수 없습니다.")

    print(f"발견된 파일 {len(files)}개 처리 중...")
    all_data, header = [], None
    for i, fname in enumerate(files):
        df = pd.read_excel(os.path.join(folder_path, fname), sheet_name=0, header=None)
        if i == 0:
            header = df.iloc[1].tolist()
        all_data.append(df.iloc[2:].reset_index(drop=True))
        print(f"  [{i+1:02d}] {fname} → {len(df)-2:,}행")

    combined = pd.concat(all_data, ignore_index=True)
    print(f"\n총 {len(combined):,}행 통합 완료")

    # 서식 적용 저장
    wb = Workbook()
    ws = wb.active
    ws.title = "도로명정보조회"
    thin = Side(style="thin")
    border = Border(left=thin, right=thin, top=thin, bottom=thin)
    hdr_font  = Font(name="Arial", bold=True, size=10, color="FFFFFF")
    hdr_fill  = PatternFill(fill_type="solid", start_color="4472C4", end_color="4472C4")
    hdr_align = Alignment(horizontal="center", vertical="center", wrap_text=True)
    dat_font  = Font(name="Arial", size=10)
    dat_align = Alignment(vertical="center")

    for c, name in enumerate(header, 1):
        cell = ws.cell(row=1, column=c, value=name)
        cell.font = hdr_font; cell.fill = hdr_fill
        cell.alignment = hdr_align; cell.border = border

    for r, row in enumerate(combined.itertuples(index=False), 2):
        for c, val in enumerate(row, 1):
            cell = ws.cell(row=r, column=c)
            cell.value = None if pd.isna(val) else (int(val) if isinstance(val, float) and val == int(val) else val)
            cell.font = dat_font; cell.alignment = dat_align; cell.border = border

    widths = [8, 12, 20, 30, 10, 8, 14, 8, 8, 10]
    for i in range(len(header)):
        ws.column_dimensions[get_column_letter(i+1)].width = widths[i] if i < len(widths) else 15
    ws.row_dimensions[1].height = 20
    ws.freeze_panes = "A2"
    wb.save(output_path)
    return output_path

result = merge("[폴더경로]", "[시도명]")
print(f"\n✅ 완료: {result}")
```

> **주의:** `[폴더경로]`와 `[시도명]`은 사용자가 제공한 실제 값으로 치환합니다.

### Step 4 — 결과 보고
- 생성된 파일 경로 안내
- 시군구별 건수 요약 (선택적)
- 오류 발생 시 원인과 해결 방법 안내
