# Midnight Galaxy (미드나이트 갤럭시)

강렬한 프레젠테이션을 위한 깊은 보라색과 신비로운 색조의 극적이고 우주적인 테마입니다.

## 색상 팔레트

- **딥 퍼플(Deep Purple)**: `#2b1e3e` - 풍부한 어두운 기본 색상
- **코스믹 블루(Cosmic Blue)**: `#4a4e8f` - 신비로운 중간 색조
- **라벤더(Lavender)**: `#a490c2` - 부드러운 강조 색상
- **실버(Silver)**: `#e6e6fa` - 밝은 하이라이트와 텍스트

## 타이포그래피

- **헤더**: FreeSans Bold
- **본문**: FreeSans

## 최적 사용 분야

엔터테인먼트 산업, 게임 프레젠테이션, 나이트라이프 장소, 럭셔리 브랜드, 크리에이티브 에이전시
