# Ocean Depths (오션 뎁스)

깊은 바다의 고요함을 연상시키는 전문적이고 차분한 해양 테마입니다.

## 색상 팔레트

- **딥 네이비(Deep Navy)**: `#1a2332` - 주요 배경 색상
- **틸(Teal)**: `#2d8b8b` - 하이라이트와 강조용 강조 색상
- **씨폼(Seafoam)**: `#a8dadc` - 밝은 요소용 보조 강조 색상
- **크림(Cream)**: `#f1faee` - 텍스트와 밝은 배경

## 타이포그래피

- **헤더**: DejaVu Sans Bold
- **본문**: DejaVu Sans

## 최적 사용 분야

기업 프레젠테이션, 재무 보고서, 전문 컨설팅 덱, 신뢰 구축 콘텐츠
