# Tech Innovation (테크 이노베이션)

최첨단 기술 프레젠테이션에 완벽한 고대비 색상의 대담하고 현대적인 테마입니다.

## 색상 팔레트

- **일렉트릭 블루(Electric Blue)**: `#0066ff` - 생동감 있는 주요 강조 색상
- **네온 시안(Neon Cyan)**: `#00ffff` - 밝은 하이라이트 색상
- **다크 그레이(Dark Gray)**: `#1e1e1e` - 깊은 배경
- **화이트(White)**: `#ffffff` - 깔끔한 텍스트와 대비

## 타이포그래피

- **헤더**: DejaVu Sans Bold
- **본문**: DejaVu Sans

## 최적 사용 분야

기술 스타트업, 소프트웨어 출시, 혁신 쇼케이스, AI/ML 프레젠테이션, 디지털 전환 콘텐츠
