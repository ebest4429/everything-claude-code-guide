# Forest Canopy (포레스트 캐노피)

울창한 숲 환경에서 영감을 받은 자연적이고 안정적인 대지 색조의 테마입니다.

## 색상 팔레트

- **포레스트 그린(Forest Green)**: `#2d4a2b` - 주요 짙은 녹색
- **세이지(Sage)**: `#7d8471` - 차분한 녹색 강조
- **올리브(Olive)**: `#a4ac86` - 밝은 강조 색상
- **아이보리(Ivory)**: `#faf9f6` - 배경과 텍스트

## 타이포그래피

- **헤더**: FreeSerif Bold
- **본문**: FreeSans

## 최적 사용 분야

환경 프레젠테이션, 지속가능성 보고서, 아웃도어 브랜드, 웰니스 콘텐츠, 유기농 제품
