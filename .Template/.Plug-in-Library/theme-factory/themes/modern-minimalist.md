# Modern Minimalist (모던 미니멀리스트)

최대한의 다용도성을 위한 세련된 그레이스케일 팔레트의 깔끔하고 현대적인 테마입니다.

## 색상 팔레트

- **차콜(Charcoal)**: `#36454f` - 주요 어두운 색상
- **슬레이트 그레이(Slate Gray)**: `#708090` - 강조용 중간 회색
- **라이트 그레이(Light Gray)**: `#d3d3d3` - 배경과 구분선
- **화이트(White)**: `#ffffff` - 텍스트와 깔끔한 배경

## 타이포그래피

- **헤더**: DejaVu Sans Bold
- **본문**: DejaVu Sans

## 최적 사용 분야

기술 프레젠테이션, 건축 포트폴리오, 디자인 쇼케이스, 현대적 비즈니스 제안서, 데이터 시각화
