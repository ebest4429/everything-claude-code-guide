# 재정분석설계 스킬 사용 예제

---

## 기본 사용법

### 1. 스킬 트리거
```
/재정분석설계
```
또는
```
/재정분석
```

### 2. Claude가 질문
Claude가 다음 정보를 순서대로 물어봅니다:
1. 지자체명
2. 분석 목적
3. 집중 분야 (선택)
4. 분석 기간 (선택)

### 3. 파일 생성
7개 파일이 자동 생성되어 제공됩니다.

---

## 예제 1: 부산광역시 투자 판단

### 실행
```
사용자: /재정분석설계

Claude: 지자체 재정 분석 프로젝트를 설계하겠습니다. 
        먼저 몇 가지 정보가 필요합니다.

사용자: 
- 지자체명: 부산광역시
- 분석 목적: 서면 재개발 지역 투자 판단
- 집중 분야: 도로/교통, 재개발
- 분석 기간: 2020-2028
```

### 출력
```
✅ 부산광역시 재정분석 프로젝트 파일 7개 생성 완료!

📁 파일 목록:
1. claude.md - 부산광역시 분석 전문가 역할 정의
2. context.md - 서면 재개발 투자 판단 목적 반영
3. rules.md - 분석 원칙
4. 분석플랜.md - 10개 세션 계획
5. workspace.md - 작업 추적 (초기화)
6. NotebookLM_프롬프트.md - 부산 데이터 추출 요청
7. Claude_세션1_시작프롬프트.md - 세션1 시작용

📋 다음 단계:
1. 위 7개 파일 다운로드
2. '부산광역시-재정분석' 폴더 생성
3. 파일 배치
4. NotebookLM에 데이터 추출 프롬프트 전달
5. Claude CLI 세션 1 시작

📚 필요한 데이터:
- 부산광역시 중기지방재정계획 (2020-2024, 2021-2025 등)
- 부산광역시 투자심사결과 (2015-2024)
- 미집행 도시계획시설 집행계획 (선택)
```

## 예제 2: 경기도 GTX 분석

### 실행
```
사용자: /재정분석

Claude: [정보 수집]

사용자:
- 지자체명: 경기도
- 분석 목적: GTX 노선별 역세권 개발 예측
- 집중 분야: 광역교통, 도시개발
- 분석 기간: 2019-2027
```

### 출력
```
✅ 경기도 재정분석 프로젝트 파일 7개 생성 완료!

파일 특징:
- 행정구역: 31개 시/군 반영
- 분석 초점: GTX-A/B/C 노선별 차이
- 지역 특성: 수도권 광역교통망 중심

📋 다음 단계: (동일)

📚 필요한 데이터:
- 경기도 중기지방재정계획
- 경기도 투자심사결과
- GTX 관련 별도 자료 (선택)
```

## 고급 사용 팁

### Tip 1: 집중 분야 복수 선택
```
집중 분야: 도로/교통, 산업단지, 재개발
```
→ 분석플랜.md에서 해당 분야 우선 분석

### Tip 2: 기간 커스터마이징
```
분석 기간: 2017-2029 (12년)
```
→ 더 긴 추이 분석 가능

### Tip 3: 특정 지역 집중
```
분석 목적: 부산 해운대구 마린시티 주변 개발 예측
```
→ context.md에 구체적 지역 명시

## 생성된 파일 활용 흐름

```
[1단계] 스킬 실행 → 7개 파일 생성
   ↓
[2단계] 로컬 폴더 구성
   {지자체명}-재정분석/
   ├── (7개 파일 배치)
   ├── data/raw/
   ├── data/processed/
   └── output/
   ↓
[3단계] NotebookLM 데이터 추출
   - NotebookLM_프롬프트.md 사용
   - 6개 데이터 세트 수신
   - data/raw/ 저장
   ↓
[4단계] Claude CLI 세션 진행
   - Claude_세션1_시작프롬프트.md 사용
   - 세션 1 → 2 → 3 → ... → 10
   - workspace.md로 진행 추적
   ↓
[5단계] 최종 보고서 완성
   - output/최종보고서_{지자체명}재정분석.md
   - PDF 변환
```

## 다른 지자체 추가 분석

### 대전 분석 후 부산 추가
```
1. 대전 분석 완료
2. /재정분석설계 다시 실행
3. "부산광역시" 입력
4. 새 폴더에 부산용 7개 파일 생성
5. 동일한 방식으로 분석 진행
```

### 비교 분석
- 대전 최종 보고서 + 부산 최종 보고서
- → Claude에게 "두 도시 비교 분석" 요청
- → 비교 리포트 작성

## 문제 해결

### Q: 지자체 정보가 없다고 나오면?
A: 지자체정보.md에 해당 지자체 추가 필요
   → skill-creator 스킬로 업데이트

### Q: 템플릿 변수가 치환 안 되면?
A: 지자체명 정확히 입력
   예: "부산" ❌ → "부산광역시" ✅

### Q: 데이터가 부족하면?
A: workspace.md에 기록 후
   분석 가능 범위 내에서 진행
   → 최종 보고서에 한계 명시

## 스킬 개선 제안

분석 진행 중 발견한 개선점은:
1. workspace.md에 메모
2. 모든 세션 완료 후
3. SKILL.md 업데이트 제안
