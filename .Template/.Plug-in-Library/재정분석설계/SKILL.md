# 재정분석 설계 스킬

## 스킬 개요

이 스킬은 지자체(광역시/도) 중기지방재정계획 분석 프로젝트를 자동으로 설계하고, 필요한 프로젝트 파일 7개와 실행 가이드를 생성합니다.

### 주요 기능
- 지자체 맞춤형 분석 프로젝트 파일 자동 생성
- NotebookLM 데이터 추출 프롬프트 제공
- Claude CLI 세션별 실행 프롬프트 생성
- 다른 지자체 분석 시 재사용 가능

### 사용 대상
- 부동산 투자자 (지역 개발 방향 예측)
- 정책 연구자 (재정 흐름 분석)
- 지역 개발 관심자

## 실행 프로세스

### 1단계: 사용자 정보 수집

다음 정보를 사용자에게 물어봅니다:

1. **지자체명** (필수)
   - 예: 부산광역시, 광주광역시, 경기도, 충청남도
   
2. **분석 목적** (필수)
   - 예: "부동산 투자 판단", "정책 연구", "지역 개발 방향 파악"
   
3. **집중 분석 분야** (선택)
   - 도로/교통 인프라
   - 산업단지/경제
   - 문화/체육 시설
   - 복지/환경
   - 전체 (기본값)

4. **분석 기간** (선택)
   - 기본값: 2019-2028 (10년)
   - 사용자 지정 가능

5. **행정구역 정보** (자동 추론 또는 사용자 입력)
   - 광역시: 구 목록 (예: 부산 - 중구, 서구, 동구, 영도구...)
   - 도: 시/군 목록 (예: 경기도 - 수원시, 성남시, 고양시...)

### 2단계: 프로젝트 파일 7개 생성

지자체명과 사용자 정보를 반영하여 다음 파일 생성:

1. `claude.md` - Claude 역할 정의 (지자체명 치환)
2. `context.md` - 프로젝트 배경 (분석 목적, 지자체 특성 반영)
3. `rules.md` - 분석 원칙 (범용, 수정 불필요)
4. `분석플랜.md` - 단계별 계획 (집중 분야 반영)
5. `workspace.md` - 작업 상태 추적 (초기화)
6. `NotebookLM_프롬프트.md` - 데이터 추출 요청 (지자체명 치환)
7. `Claude_세션1_시작프롬프트.md` - 세션1 시작용

### 3단계: 파일 제공 및 안내

- 7개 파일을 다운로드 가능하게 제공
- 다음 단계 실행 가이드 제공
- 데이터 준비 체크리스트 제공

## 템플릿 변수 치환 규칙

### 기본 치환
- `{{지자체명}}`: 사용자 입력 지자체명
- `{{분석목적}}`: 사용자 입력 분석 목적
- `{{행정구역목록}}`: 구/시/군 목록
- `{{분석기간시작}}`: 예: 2019
- `{{분석기간종료}}`: 예: 2028

### 지자체 유형별 맞춤
- **광역시**: 구 중심 분석
- **도**: 시/군 중심 분석
- **특별시/특별자치시**: 자치구 중심

### 예시
```
대전광역시 → 부산광역시
5개 구 (동구, 중구, 서구, 유성구, 대덕구) 
  → 16개 구 (중구, 서구, 동구, 영도구, 부산진구...)
```

### 사용 사례 1: 부산 분석
```

Claude: 
1. 지자체명: 부산광역시
2. 분석 목적: 서면 재개발 투자 판단
3. 집중 분야: 도로/교통, 재개발
4. 기간: 2020-2028

→ 부산 맞춤 7개 파일 생성

Claude:

2. 분석 목적: GTX 노선 주변 개발 예측

4. 기간: 2019-2027

→ 경기도 맞춤 7개 파일 생성 (31개 시/군 반영)

## 출력 구조

### 폴더 구조 안내
```
{지자체명}-재정분석/
├── claude.md
├── context.md
├── rules.md
├── 분석플랜.md
├── workspace.md
├── NotebookLM_프롬프트.md
├── Claude_세션1_시작프롬프트.md
└── data/
    ├── raw/          ← NotebookLM 데이터 저장
    └── processed/    ← 중간 처리 데이터
└── output/           ← 세션별 분석 결과
```

```

3. 파일 배치
4. NotebookLM에 데이터 추출 프롬프트 전달
5. Claude CLI로 세션 1 시작

📚 필요한 데이터:
- {지자체명} 중기지방재정계획 (최근 2-3개년)

## 주의사항

### 지자체별 차이점
- **광역시**: 구 단위 분석 (5-16개 구)
- **도**: 시/군 단위 분석 (10-31개 시/군)
- **특별시**: 자치구 단위 (25개 자치구)
- **특별자치시/도**: 읍/면/동 또는 행정시 단위

### 데이터 가용성
- 지자체마다 공개 범위 상이
- 일부 지자체는 투자심사결과 미공개
- 이 경우 중기재정계획만으로 제한적 분석

### 분석 한계
- 민간 투자 미포함 (공공 재정만)
- 계획 vs 실제 집행 차이 존재
- 정책 변경 가능성 항시 존재
