# 재정분석설계 스킬

## 개요

지자체 중기지방재정계획 분석 프로젝트를 자동으로 설계하는 스킬입니다.

### 주요 기능
- 지자체 맞춤형 분석 프로젝트 파일 7개 자동 생성
- 대전, 부산, 서울 등 모든 광역 지자체 지원
- NotebookLM 연계 데이터 추출 자동화
- Claude CLI 세션별 프롬프트 제공

---

## 빠른 시작

```
/재정분석설계
```

1. 지자체명 입력 (예: 부산광역시)
2. 분석 목적 입력 (예: 투자 판단)
3. 7개 파일 다운로드
4. NotebookLM 데이터 추출
5. Claude CLI 분석 시작

---

## 파일 구조

```
/mnt/skills/user/재정분석설계/
├── SKILL.md              # 스킬 정의 (트리거, 프로세스)
├── README.md             # 이 파일
├── 예제.md               # 사용 예제
├── 지자체정보.md          # 지자체별 데이터
└── templates/            # 템플릿 파일들
    ├── claude.md
    ├── context.md
    ├── rules.md
    ├── 분석플랜.md
    ├── workspace.md
    ├── NotebookLM_프롬프트.md
    └── Claude_세션1_시작프롬프트.md
```

## 지원 지자체

### 광역시
- 서울특별시
- 부산광역시, 대구광역시, 인천광역시
- 광주광역시, 대전광역시, 울산광역시

### 특별자치시/도
- 세종특별자치시
- 제주특별자치도
- 강원특별자치도, 전북특별자치도

### 도
- 경기도
- 충청북도, 충청남도
- 전라남도
- 경상북도, 경상남도

총 **17개 광역 지자체** 지원

## 템플릿 변수

생성되는 파일들에서 자동 치환되는 변수:

- `{{지자체명}}`: 사용자 입력 지자체명
- `{{분석목적}}`: 사용자 입력 분석 목적
- `{{행정구역유형}}`: 구/시/군/자치구
- `{{행정구역목록}}`: 해당 지자체의 구/시/군 목록
- `{{행정구역예시}}`: 대표 행정구역 1개
- `{{분석기간시작}}`, `{{분석기간종료}}`: 분석 기간
- `{{지자체특성}}`: 해당 지자체의 특징
- `{{지자체특성_분석아이디어}}`: 분석 힌트
- `{{지자체특성_주의사항}}`: 주의사항
- `{{지자체특성_질문}}`: 맞춤 질문

## 작동 원리

### 1. 사용자 입력 수집
```
ask_user_input 도구 사용
→ 지자체명, 분석목적, 집중분야, 분석기간
```

### 2. 지자체 정보 조회
```
지자체정보.md에서 해당 지자체 데이터 로드
→ 행정구역, 특성 등
```

### 3. 템플릿 변수 치환
```
templates/ 폴더의 7개 파일 읽기
→ {{변수}} 치환
→ 최종 파일 생성
```

### 4. 파일 제공
```
create_file 도구로 7개 파일 생성
→ present_files로 다운로드 제공
```
