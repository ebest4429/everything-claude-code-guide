# 분석 원칙 및 제약사항

## 데이터 해석 기준

### 사업 생애주기 판단

#### 1단계: 계획 수립
**포착 방법**:
- 투자심사 결과에서 "예비타당성조사 대상" 선정
- 중기재정계획에서 첫 예산 배정 (보통 예타비)

**시그널**:
- 예타비 수억~수십억 규모
- "타당성 조사", "기본계획 수립" 등의 사업명

#### 2단계: 초기 투입 (예타/사전조사)
**판단 기준**:
- 총사업비의 1-5% 수준 예산
- 1-2년간 집중 투입 후 일시 중단

**의미**:
- 사업 실현 가능성 검토 중
- 본격 투입 여부 미결정

#### 3-A단계: 본격 추진
**판단 기준**:
- 예산 연속 배정 (3년 이상)
- 매년 증가 또는 유지
- 총사업비 10% 이상 연간 투입

**의미**:
- 사업 확정
- 완료까지 지속 투입 예상

#### 3-B단계: 정체기
**판단 기준**:
- 예타 통과 후 2년 이상 예산 미배정
- 또는 연 10억 미만 소액 배정

**의미**:
- 재검토 중
- 우선순위 하락
- 백지화 가능성

#### 4단계: 완료
**판단 기준**:
- 예산 급감 (전년 대비 50% 이상 감소)
- 이후 1-2년 내 예산 0

**의미**:
- 사업 종료
- 마무리 작업 중

---

### 우선순위 상승 판단 (3가지 유형)

#### 유형 1: 증가율형
**기준**: 전년 대비 **50% 이상** 예산 증가

**계산 예시**:
```
2022년: 50억
2023년: 80억
증가율 = (80-50)/50 × 100 = 60% → 증가율형 ✅
```

**의미**:
- 기존 사업 가속화
- 정책 우선순위 상승

**주의**:
- 기저효과 (예: 10억 → 20억도 100% 증가지만 절대액 작음)

#### 유형 2: 절대액형
**기준**: 전년 대비 **50억원 이상** 예산 증가

**계산 예시**:
```
2022년: 200억
2023년: 260억
증가액 = 60억 → 절대액형 ✅
```

**의미**:
- 대규모 재정 투입 결정
- 사업 단계 전환 (예: 설계 → 공사)

**우선 적용**:
- 대형 사업은 증가율보다 절대액이 의미 있음

#### 유형 3: 신규형
**기준**: 과거 3년간 예산 0 → 신규 배정

**확인 사항**:
```
2020년: 0
2021년: 0
2022년: 0
2023년: 30억 → 신규형 ✅
```

**의미**:
- 새로운 정책 방향
- 신규 수요 발생
- 타 지역 사례 벤치마킹

**주의**:
- 3-B단계(정체기) 후 재개도 신규형으로 분류

---

### 정체 판단 기준

| 상황 | 판단 | 조치 |
|------|------|------|
| **3년 연속 예산 0** | 사실상 백지화 가능성 | 투자 대상에서 제외 고려 |
| **예타 후 2년 이상 연 10억 미만** | 재검토 또는 우선순위 하락 | 주의 관찰 필요 |
| **2년 연속 50% 이상 감소** | 사업 축소/종료 | 투자 회수 신호 |
| **총사업비 대비 5% 미만 진척 (5년 경과)** | 진척 부진 | 실현 가능성 의심 |

## 분석 방법론

### 연도별 변화 추이 분석 (최우선)

#### 기본 프로세스
1. 사업별 2019-2028 예산 추이표 작성
2. 증감 패턴 분류:
   - 지속 상승형
   - 급증형
   - 등락형
   - 정체형
   - 감소형
3. 패턴별 의미 해석

#### 시각화 예시
```
예산(억원)
    │
150 │         ┌───┐
    │         │   │
100 │     ┌───┘   └───┐
    │     │           │
 50 │ ┌───┘           └───
    │ │
  0 └─┴───┴───┴───┴───┴───┴───
    19  20  21  22  23  24  25
```

### 사업 간 비교 분석

#### 동일 분야 내 비교
예: 도로 사업 간 우선순위

| 사업명 | 2023 예산 | 2024 예산 | 증가율 | 순위 변화 |
|--------|-----------|-----------|--------|-----------|
| A도로 | 100억 | 150억 | +50% | ↑ |
| B도로 | 80억 | 60억 | -25% | ↓ |

#### 구별 비교
예: 5개 구의 도로 예산 배분

### 구별 비교 분석

#### 절대액 vs 증가율 병행
- 절대액: 투자 규모
- 증가율: 정책 관심도

## 수치 처리 원칙

### 단위 통일
- **기본 단위**: 억원
- 백만원 → 억원 환산 (÷100)
- 천원 단위 절사

### 증감률 계산
```
증가율(%) = (금년 - 전년) / 전년 × 100
```

**소수점**: 첫째 자리까지 (반올림)

### 비교 시점
- **전년 대비**: 기본
- **3년 평균 대비**: 변동성 큰 경우
- **2019년 대비**: 장기 추이

```
(출처: investment2021.pdf, 중기재정계획 2019-2023)
```

## 해석 원칙

### 인과관계 vs 상관관계
❌ "예산 증가 **때문에** 개발이 빠르다"
✅ "예산 증가 **경향을 보이며** 개발 진척이 예상된다"

### 단정 vs 추정
❌ "이 지역은 **반드시** 개발된다"
✅ "이 지역은 재정 투입 흐름상 개발 **가능성이 높다** (근거: ...)"

### 예외사항 병기
```
유성구 도로 예산 2021-2023 연평균 30% 증가.
다만, 2022년 일시 감소는 세종-대전 광역도로 중앙정부 이관으로 추정.
```

## 금지사항

### 1. 근거 없는 추측 금지
❌ "시장 의지로 보아 ..."
❌ "정치적 이유로 ..."
❌ "선거 때문에 ..."

✅ 오직 재정 데이터로만 판단

### 2. 과도한 일반화 금지
❌ 1개 사업 예산 증가 → "○○구 전체가 우선순위"
✅ 여러 사업 종합 후 판단

### 3. 맥락 없는 수치 나열 금지
❌ 표만 제시하고 끝
✅ 표 + 해석 + 의미

### 4. 확정적 미래 예측 금지
❌ "2026년 ○○ 완공"
✅ "계획상 2026년 완공 예정, 다만 예산 변동 가능성 있음"

## 불확실성 표현

### 신뢰도 3단계

| 신뢰도 | 표현 | 조건 |
|--------|------|------|
| **높음** | "~로 판단됨", "명확히" | 3개 이상 데이터 포인트 일치 |
| **중간** | "~로 추정됨", "가능성 있음" | 2개 데이터 포인트 또는 간접 근거 |
| **낮음** | "~일 수 있음", "불확실" | 1개 데이터 또는 추론 |

### 예외/변수 명시
```
다만, 다음 변수로 인해 변경 가능:
- 중앙정부 정책 변화
- 경제 상황 악화
- 지역 민원 등
```
