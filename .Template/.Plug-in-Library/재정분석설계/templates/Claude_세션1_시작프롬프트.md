   - 신설 (새로 만드는 도로)

   - 정비 (보수, 개선)

### Step 1-3: 예산 추이 분석
각 도로 사업의 {{분석기간시작}}-{{분석기간종료}} 예산 흐름을 분석하여 다음으로 분류:

#### A. 지속 상승형
```
예: {{분석기간시작}}(10억) → {{분석기간시작+1}}(15억) → {{분석기간시작+2}}(20억)
→ 연속적 증가, 본격 추진 중
```

#### B. 급증형
**증가율형**: 전년 대비 50% 이상 증가
**절대액형**: 전년 대비 50억원 이상 증가

#### C. 신규형
```
예: {{분석기간시작}}-{{분석기간시작+2}}(0억) → {{분석기간시작+3}}(30억) [신규 편성]
```

#### D. 정체형
```
예: {{분석기간시작}}(예타 5억) → {{분석기간시작+1}}-{{분석기간시작+3}}(0억)
→ 예타 후 장기 미집행
```

#### E. 감소/종료형
```
예: {{분석기간시작+2}}(50억) → {{분석기간시작+3}}(20억) → {{분석기간시작+4}}(5억)
→ 사업 마무리 단계
```
