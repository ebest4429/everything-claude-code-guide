## 🔍 추출 시 유의사항

1. **정확한 숫자**: 예산 단위(백만원/억원) 명확히 표시
2. **출처 병기**: 각 데이터가 어느 문서 몇 페이지인지 기록
3. **중복 제거**: 동일 사업이 여러 문서에 있으면 최신 데이터 우선
4. **빈 칸 처리**: 데이터 없으면 "-" 표시, 0이면 "0" 표시
5. **일관성**: 사업명이 문서마다 다르면 통일된 명칭으로 정리
