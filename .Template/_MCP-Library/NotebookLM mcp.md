# notebooklm MCP

## 용도
NotebookLM 노트 연동. 문서 분석, 리서치 작업 시 사용.

## 적합한 프로젝트
- 문서 기반 프로젝트
- 리서치/분석 작업

## 현재 검증 버전
`1.2.1` (검증일: 2025-03-13)

## 설정값 복사
```json
"notebooklm": {
  "command": "cmd",
  "args": [
    "/c",
    "npx",
    "-y",
    "notebooklm-mcp@1.2.1"
  ]
}
```

## 업데이트 이력
- 1.2.1 : 안정화 버전 확인
