# memory MCP

## 용도
대화 간 컨텍스트(엔티티, 관계, 메모)를 영속적으로 저장.
세션이 종료되어도 프로젝트 지식, 코딩 컨벤션, 결정 사항 등을 기억.

## 적합한 프로젝트
- 장기 진행 프로젝트
- 여러 세션에 걸쳐 작업이 이어지는 경우
- **전역 등록 강력 권장** (프로젝트 무관하게 항상 활성화)

## API 키
불필요

## 현재 검증 버전
`@latest` — 공식 Anthropic 패키지

## 설정값 복사

```json
"memory": {
  "command": "cmd",
  "args": [
    "/c",
    "npx",
    "-y",
    "@modelcontextprotocol/server-memory"
  ]
}
```

## 주의사항
- 잘못된 정보가 저장되면 이후 세션에서 지속 오염 → **주기적 정리 필요**
- 세션 시작 시 `"현재 저장된 memory 목록 보여줘"` 로 상태 확인 권장
- 저장 권장: 코딩 컨벤션, 기술 스택, API 엔드포인트
- 저장 비권장: 임시 결정, 실험 중인 내용, 작업 진행 상태
