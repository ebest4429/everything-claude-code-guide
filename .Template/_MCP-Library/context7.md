# context7 MCP

## 용도
라이브러리/프레임워크의 최신 공식 문서를 실시간으로 Claude에게 제공.
Claude 학습 데이터의 구버전 정보 대신 현재 문서 기준으로 답변 유도.

## 적합한 프로젝트
- 빠르게 버전업되는 프레임워크 사용 프로젝트 (Next.js, React, Vue 등)
- 외부 라이브러리 의존도가 높은 프로젝트
- 전역 등록 권장

## API 키
불필요

## 현재 검증 버전
`@latest` — Upstash 제공, 문서 DB가 지속 업데이트되므로 latest 유지

## 설정값 복사

```json
"context7": {
  "type": "stdio",
  "command": "cmd",
  "args": [
    "/c",
    "npx",
    "-y",
    "@upstash/context7-mcp"
  ],
  "env": {}
}
```

## 사용 팁
- 자동 호출되기도 하지만 **명시적 요청이 품질 보장**에 유리
- 프롬프트에 `"use context7"` 추가 시 강제 최신 문서 기준 답변
- 버전이 중요한 라이브러리 작업 시 반드시 명시
