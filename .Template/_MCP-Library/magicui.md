# magicui MCP

## 용도
Magic UI 컴포넌트 라이브러리 연동. 애니메이션, 인터랙티브 UI 컴포넌트 조회 및 설치 자동화.

## 적합한 프로젝트
- Magic UI 기반 프론트엔드 프로젝트
- 고품질 UI 애니메이션이 필요한 프로젝트
- shadcn/ui와 병행 사용 가능

## API 키
불필요

## 현재 검증 버전
`@magicuidesign/mcp@latest`

## 설정값 복사

```json
"@magicuidesign/mcp": {
  "command": "npx",
  "args": [
    "-y",
    "@magicuidesign/mcp@latest"
  ]
}
```

## cmd 래퍼 적용 버전 (Windows 권장)

```json
"magicui": {
  "command": "cmd",
  "args": [
    "/c",
    "npx",
    "-y",
    "@magicuidesign/mcp@latest"
  ]
}
```

## 주의사항
- 비프론트엔드 프로젝트에 등록 불필요
- shadcn MCP와 함께 사용 시 UI 컴포넌트 선택 범위 확대
